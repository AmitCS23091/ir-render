What is Information Retrieval?

The Definition

Information Retrieval (IR) is the process of obtaining structure or unstructured information resources (usually documents) relevant to an information need from a large collection.

Real-Life Examples:

  • Google Web Search
  • YouTube Video Search
  • Amazon Product Search

1. Document Collection

Gathering all possible data (web pages, files, images) into a massive database.

2. Preprocessing

Cleaning the text so the computer understands it better.

  • Tokenization: Breaking text into single words (tokens).
  • Stopword Removal: Removing common words like "the", "is", "at".
  • Stemming: Reducing words to their root form (e.g., "running" → "run").

3. Indexing

Creating an organized "dictionary" (Inverted Index) showing exactly which document contains which word to speed up searching.

4. Query Processing

When a user types a search, the system cleans the query exactly like it cleaned the documents.

5. Ranking Results

Applying mathematical models to score documents based on relevance and showing the highest-scoring ones first.

The Core IR Models

How systems map queries to documents mathematically

Boolean Model

Uses Exact Match based on Boolean logic (AND, OR, NOT).

Example:
"Apple AND Phone NOT Fruit"
Clean and simple logic
No partial matches or ranking

Vector Space Model

Represents documents and queries as vectors in a multi-dimensional space.

Example:
Uses Cosine Similarity to find angle between query and document vectors.
Excellent for ranking results
Assumes terms are independent

Probabilistic Model

Estimates the probability that a document will be relevant to a particular query.

Example:
BM25 algorithm (Used by modern search engines like Elasticsearch).
Mathurally rigorous & accurate
Hard to estimate initial probabilities

Real-World Applications

Web Search Engines

Crawling and indexing billions of web pages to serve user queries in milliseconds.

E-Commerce

Finding the exact product among millions of items (like Amazon or Flipkart).

Media Recommendations

Recommending movies and music based on user profiles and tags (Netflix, Spotify).

Chatbots / AI

Retrieving relevant knowledge base articles to answer customer questions automatically.
